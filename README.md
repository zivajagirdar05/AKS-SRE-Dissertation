## Dissertation Objective

This research investigates how Site Reliability Engineering (SRE) practices improve the reliability, availability, observability, and operational efficiency of Azure Kubernetes Service (AKS).

The study includes:

- Literature Review
- AKS Architecture
- Azure Monitor
- Prometheus
- Grafana
- Kubernetes
- Docker
- Reliability Metrics
- SLOs
- SLIs
- Error Budgets
- Experimental Evaluation
